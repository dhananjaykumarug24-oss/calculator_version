# calculator_version
## v1.0 – Basic Calculator
- Added a simple basic calculator
- Performs basic operations (+, −, ×, ÷)
- Runs in terminal

## v2.0 – GUI Calculator
- Added a GUI calculator
- User-friendly interface with buttons

## v3.0 – Web Calculator
- Developed a web-based calculator
- Runs in browser

## v4.0 – Scientific Calculator
- Added GUI Scientific Calculator
- Includes advanced features:

  - Trigonometric functions (sin, cos, tan)
  - Logarithmic functions (log, ln)
  - Power functions (x², xʸ)
  - Factorial, percentage, memory functions
- Improved UI design

# How to Run the Project

## 1. Clone the Repository
- git clone <your-repo-link>
- cd calculator_version

## 2. Run Version 1.0 (Basic Calculator)
- python3 calculator_v1.py

## 3. Run Version 2.0 (GUI Calculator)
- python3 calculator_v2.py

## 4. Run Version 3.0 (Web Calculator)
### Install Flask:
- pip install flask

### Run:
- python3 calculator_v3/app.py
- Open browser:
http://127.0.0.1:5000/

## 5. Run Version 4.0 (Scientific Calculator)
- python3 calculator_v4.py
